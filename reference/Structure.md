# Definition

When we say something has structure, what does that mean?

It means:

It is not random.

## The Cleanest Possible Definition

Structure =
Non-random organization of parts within a boundary.

Expanded slightly:

Structure means:
1. There are parts.
2. The parts are arranged in some way.
3. The arrangement follows some rule (not arbitrary).
